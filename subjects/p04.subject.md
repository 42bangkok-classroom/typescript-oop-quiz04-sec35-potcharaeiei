# โจทย์ที่ 4: ดึงข้อมูล User ตาม ID และ Filter Fields

**Path:** `GET /users/:id`

**Query Parameters:** `fields` (string, comma-separated) - ระบุ fields ที่ต้องการดึง เช่น `firstName,lastName,email`

**ผลลัพธ์:** ข้อมูล user ตาม id ที่ระบุ และแสดงเฉพาะ fields ที่ระบุใน query parameter (ถ้ามี)

**ไฟล์ที่ต้องแก้ไข (ไฟล์ที่มีอยู่แล้วจากข้อ 2-3):**

- `src/user/user.service.ts` (เพิ่ม method `findOne(id: string, fields?: string[])`)
- `src/user/user.controller.ts` (เพิ่ม endpoint `GET /users/:id` พร้อมรับ query parameter `fields`)

**ตัวอย่างการเรียก API:**

1. ดึงข้อมูล user ทั้งหมดตาม id:

```bash
curl http://localhost:3000/users/1
```

```json
{
  "id": "1",
  "firstName": "สมชาย",
  "lastName": "ใจดี",
  "email": "somchai.jaidee@example.com",
  "username": "somchai_j"
}
```

2. ดึงเฉพาะ fields ที่ระบุ:

```bash
curl http://localhost:3000/users/1?fields=firstName,lastName,email
```

```json
{
  "firstName": "สมชาย",
  "lastName": "ใจดี",
  "email": "somchai.jaidee@example.com"
}
```

3. ไม่พบ user (id):

```bash
curl http://localhost:3000/users/999
```

```json
{
  "statusCode": 404,
  "message": "User not found",
  "error": "Not Found"
}
```

**หมายเหตุ:**

- ถ้าไม่พบ user ตาม id ที่ระบุ ให้ throw `NotFoundException` (404 status code)
- Query parameter `fields` เป็น optional (ถ้าไม่ระบุให้แสดงข้อมูลทั้งหมด)
- Fields ที่ระบุต้องเป็น comma-separated string เช่น `firstName,lastName,email`
