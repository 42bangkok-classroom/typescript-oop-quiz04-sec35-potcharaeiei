# โจทย์ที่ 3: สร้าง Interface User และอ่านข้อมูลจากไฟล์

**Path:** `GET /users`

**ผลลัพธ์:** ข้อมูล users ทั้งหมดจากไฟล์ `data/users.json`

**ไฟล์ที่ต้องสร้าง:**

- `src/user/user.interface.ts` (interface `IUser`)

**ไฟล์ที่ต้องแก้ไข (ไฟล์ที่มีอยู่แล้วจากข้อ 2):**

- `src/user/user.service.ts` (เพิ่ม method `findAll()` เพื่ออ่านข้อมูลจาก `data/users.json`)
- `src/user/user.controller.ts` (เพิ่ม endpoint `GET /users`)

**ตัวอย่างการเรียก API:**

```bash
curl http://localhost:3000/users
```

**ตัวอย่างผลลัพธ์:**

```json
[
  {
    "id": "1",
    "firstName": "สมชาย",
    "lastName": "ใจดี",
    "email": "somchai.jaidee@example.com",
    "username": "somchai_j"
  },
  {
    "id": "2",
    "firstName": "สมหญิง",
    "lastName": "รักเรียน",
    "email": "somying.rakrian@example.com",
    "username": "somying_r"
  }
  // ... (ข้อมูล users ทั้งหมด 10 รายการ)
]
```

**หมายเหตุ:**

- Interface IUser ต้องมี attributes ตามโครงสร้างข้อมูลใน `data/users.json`
- Method `findAll()` ต้องอ่านข้อมูลจากไฟล์ `data/users.json` และ return array ของ IUser objects
